一般说明

  • mean：常量固定值
  • dev：倍率
  • 1 秒 = 60 帧

导出

  • Window -> Recorder
  • 可以选择导出的分辨率、格式、图层、帧动画

Basic Settings 基本设置

  • visibility：显示/隐藏
  • name：图层名
  • spawn count：生成的图层数量
  • infinite：无限数量
  • inherit position：继承坐标
  • inherit rotation：继承旋转
  • inherit scale：继承大小
  • destory after time：生命周期结束后清除对象
  • destory with parent：将父类一起清除
  • destory when no：？
  • time to live：生命周期（帧）
  • spawn rate：生命周期 / 这个值 = 实际生命周期；千万不要设为 0，会瞬间生成一车粒子电脑直接爆炸
  • initial delay：特效开始播放时间（帧）

Position 坐标

  • set position：常规坐标
  • pva：递增坐标
  • easing：时间轴坐标
  • f-curve：？

Rotation 旋转

  • fixed angle：常规角度
  • pva：递增角度
  • easing：时间轴角度

Base Render Settings 贴图渲染设置

  • texture：贴图
  • fade-in：淡入
  • fade-out：淡出
  • frame count：淡出淡入的值

blend 颜色叠加方式，类似 PS

  • opacity：无透明度
  • blend：有透明度
  • additive：明度叠加
  • subtract：减去
  • multiply：叠加

UV 动画类型

  • standard：静态
  • scroll：循环滚动

Render Settings 渲染扩展设置

sprite 精灵

  • vertex color 自定义颜色
  • vertex coords 自定义贴图形状

color all 颜色叠加

  • fixed：固定值
  • random：随机范围
  • easing：时间轴范围
  • f-curve：？

configuration 配置

  • Billboard：固定面向镜头
  • Rotated Billboard：支持旋转

ring 光环

shape 形状

  • dount：圆形
  • crescent：半圆
