【Web】ES 6-11 语法

发表于 2020-07-09 更新于 2020-07-22 分类于 Web

点击查看更多

let

  • let 是块级作用域，属于局部变量
  • let 不会代码升级，不像 var 先引用再定义变量时会隐式的定义变量
  • var 的变量是全局的，包括常见的 for 循环里的 var i = 0。使用完毕后并不会重置，所以要将 var 改成 let

const

  • 常量使用大写
  • 块级作用域
  • 常量数组允许修改，因为引用的是同一个内存地址

变量解构赋值

  • 相当于为数组定义 key
  • 方法名可以从 play: function(){} 简化为 play(){}

模板字符串

  • 类似 C# 里的 $“你的名字是：{Name}”
  • 注意这里使用的是反引号，键盘上的波浪线

简化对象

  • 将 str : str 精简为 str

箭头函数

  • 没有构造方法
  • 没有 arguments 变量
  • 箭头函数不适合在对象的方法和涉及 this 回调的情况下使用
  • 语法上类似 C# 匿名委托
  • this 在箭头函数下的作用域指向问题

函数参数默认值

  • 类似 C# 的默认参数一样，有默认值的参数要放到最后
  • 允许用冒号为指定的参数赋值

rest 参数

  • 这是一个取代 arguments 的关键字
  • rest 参数必须放到最后
  • 它能将数组拆分成单个方法参数逐个传递
  • 数组合并和克隆

Symbol

  • 第七种数据类型（undefined string symbol object null number bool）
  • 值唯一
  • 无法与包含自身在内的其它类型运算
  • 无法使用 for 遍历，但可以用 Reflect.ownKeys 获取所有 key
  • 一般是某些关键字的内置属性，例如数组 Symbol.iterator
  • 类似 C# 里的扩展方法

迭代器

for…of 与 for…in 的区别

让对象可以使用 for…of

  • 为不可遍历的对象添加一个 symbol.iterator 方法，此方法有固定格式

生成器，类似 C# enumerable

  • 注意方法定义时要加 * 号
  • 调用只能通过 xx.next() 方法迭代
  • next() 会同时调用其它代码

Promise

  • Promise 有两个固定参数 resolve 成功 / reject 出错
  • then 提供接收成功和失败信息的方法

链式回调

Set 集合

  • 自带去重
  • 实现了 iterator

Map 键值对

  • 实现了 iterator

Class 类

  • 方法重写：子类的方法与父类的方法同名时会覆盖父类方法达成方法重写

静态成员

  • 与 C# 的概念相同，实例成员用实例类调用，静态成员直接用类调用。因为它们分别存在不同的内存区域
